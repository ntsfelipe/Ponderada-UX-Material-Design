O aplicativo foi desenvolvido utilizando React Native, Expo e React Native Paper seguindo os princípios do Material Design 3. O foco do projeto foi criar uma interface moderna, organizada e intuitiva para acompanhamento de hábitos saudáveis. Foram utilizados componentes visuais como Cards, Chips, FAB, ProgressBar e Snackbar para melhorar a experiência do usuário e fornecer feedback visual imediato. Recursos como humor do dia, hidratação e streak diário aumentam o engajamento e tornam a experiência mais interativa e gamificada.
